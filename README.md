# PronunCo Clinic Audio

Canonical clinic audio manifests and approved human voice contributor records for PronunCo.

## Purpose

This repo stores:
- audio pack manifests
- contributor metadata
- recording workflow notes
- canonical source text references for cloud TTS or human recording

Runtime audio binaries should ultimately live in object storage or CDN, not GitHub raw delivery.
